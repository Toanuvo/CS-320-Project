Overview
The home page is the first page you'll see when using this app, it displays the tasks due for the day or alternatively you can change it to display the top 5 priority tasks. The home page also great you with your desired username and you can complete tasks there as well
Task Completion
to complete tasks on the home page a complete task button similar to the task page will appear next to the task. Pressing this button will complete the task and remove it from the tasks list awarding points and updates the streak counter if required
